Class StringBuilder

A dynamic string builder for efficient char manipulation.

  • Stores data in a contiguous ArrayBuffer not unlike C++ strings or Java/C# StringBuilder.
  • Provides optimization buffer for rapid repetitive prepends
  • All arguments representing index may use negative numbers like in Python.

  • Write data directly into the buffer, overwriting any underlying data. If write exceeds the bufferLength, it will automatically expand.

      index to start writing to

    • strOrArray: string | ArrayLike<number> | ArrayLike<string>

    Returns StringBuilder

    This StringBuilder for chained calls.

    Simple write

    const sb = new StringBuilder("012345");

    sb.write(2, "abc");

    console.log(sb); // "01abc5"

    Overflow is okay

    const sb = new StringBuilder("012345");

    sb.write(5, "abc");

    console.log(sb); // "01234abc"
